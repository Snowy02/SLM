This is a classic object-oriented programming exercise designed to teach you about **abstract classes, inheritance, and method overriding** in C#.

Here is a step-by-step guide on how to set this up in **Visual Studio 2022**.

### Step 1: Create the Project
1. Open **Visual Studio 2022**.
2. Click **"Create a new project"**.
3. Search for **"Console App"** (make sure it's the one for C#). Click **Next**.
4. Name your project `RetailDeliveryCalculator`. Click **Next** and then **Create**.

---

### Step 2: Create the Class Files
In your **Solution Explorer** (the panel on the right), right-click on the project name (`RetailDeliveryCalculator`), select **Add** > **Class...**. 

You need to create **five** new class files. Name them exactly as follows:
1. `Delivery.cs`
2. `StandardDelivery.cs`
3. `ExpressDelivery.cs`
4. `InternationalDelivery.cs`
5. `FreeDelivery.cs`

---

### Step 3: Copy and Paste the Code
Replace the entire contents of each file with the code below.

#### 1. `Delivery.cs` (The Abstract Base Class)
```csharp
namespace RetailDeliveryCalculator
{
    // An abstract class cannot be instantiated directly. 
    // It provides a template for child classes.
    public abstract class Delivery
    {
        // Property to store the order value
        public double OrderAmount { get; set; }

        // Abstract method: Every child class MUST provide its own implementation
        public abstract double CalculateDeliveryCharge();

        // Concrete method: All child classes share this common display logic
        public void DisplayTotalBill()
        {
            double deliveryCharge = CalculateDeliveryCharge();
            double totalBill = OrderAmount + deliveryCharge;

            Console.WriteLine($"{this.GetType().Name}: Total Bill = {totalBill}");
            // Optional detailed view:
            // Console.WriteLine($"{this.GetType().Name} -> Order: ₹{OrderAmount}, Delivery: ₹{deliveryCharge}, Total: ₹{totalBill}");
        }
    }
}
```

#### 2. `StandardDelivery.cs`
```csharp
namespace RetailDeliveryCalculator
{
    public class StandardDelivery : Delivery
    {
        public override double CalculateDeliveryCharge()
        {
            return 50; // Fixed charge of ₹50
        }
    }
}
```

#### 3. `ExpressDelivery.cs`
```csharp
namespace RetailDeliveryCalculator
{
    public class ExpressDelivery : Delivery
    {
        public override double CalculateDeliveryCharge()
        {
            return 100; // Fixed charge of ₹100
        }
    }
}
```

#### 4. `InternationalDelivery.cs`
```csharp
namespace RetailDeliveryCalculator
{
    public class InternationalDelivery : Delivery
    {
        public override double CalculateDeliveryCharge()
        {
            return OrderAmount * 0.10; // 10% of order amount
        }
    }
}
```

#### 5. `FreeDelivery.cs`
```csharp
namespace RetailDeliveryCalculator
{
    public class FreeDelivery : Delivery
    {
        public override double CalculateDeliveryCharge()
        {
            // Charge ₹0 if amount >= 2000, otherwise ₹50
            if (OrderAmount >= 2000)
                return 0;
            else
                return 50;
        }
    }
}
```

---

### Step 4: Setup the Main Program (`Program.cs`)
Open your `Program.cs` file. Delete everything inside it and paste this code:

```csharp
using RetailDeliveryCalculator;

Console.WriteLine("--- Output for Order Amount ₹1000 ---");
RunCalculator(1000);

Console.WriteLine("\n--- Output for Order Amount ₹2500 ---");
RunCalculator(2500);

void RunCalculator(double amount)
{
    // Create objects for each delivery type
    // We can treat them all as "Delivery" objects because of Polymorphism
    Delivery standard = new StandardDelivery { OrderAmount = amount };
    Delivery express = new ExpressDelivery { OrderAmount = amount };
    Delivery international = new InternationalDelivery { OrderAmount = amount };
    Delivery free = new FreeDelivery { OrderAmount = amount };

    // Call the display method for each
    standard.DisplayTotalBill();
    express.DisplayTotalBill();
    international.DisplayTotalBill();
    free.DisplayTotalBill();
}

// Keep console window open
Console.ReadLine();
```

---

### Step 5: Run the Project
Press **F5** or the green **"Start"** arrow at the top of Visual Studio.

### What you should notice:
* **Abstract Class:** You defined the general behavior in `Delivery`, but didn't say *how* to calculate charges there.
* **Overriding:** Each child class has its own logic for `CalculateDeliveryCharge`. 
* **FreeDelivery Logic:** Notice how when the amount is ₹1000, `FreeDelivery` adds ₹50, but when the amount is ₹2500, it adds ₹0. This is because of the logic you wrote inside that class.
* **Polymorphism:** In `Program.cs`, even though we have 4 different types of classes, we can call `DisplayTotalBill()` on all of them because they all inherit from the same parent.
